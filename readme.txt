Tools, die im Verzeichnis zu installieren sind, für reibungslose Ausführung:
- VS Code (oder irgendeine andere IDE)
- Node.js (https://nodejs.org/en) als Laufzeitumgebung von JavaScript
- Express (npm install express) (npm=node package manager) für die Funktionalitäten der API
- nodemon (npm install nodemon), um Server nach Änderungen automatisch neu zu starten
- Postman (https://www.postman.com/downloads/), um mit API zu interagieren und Daten im Body anzugeben
